# 👕 게임 아바타 트랜드 검색 및 리뷰 서비스

>현재 가장 인기많은 아바타를 검색 및 착용 이미지를 기록할수 있는 웹 서비스

## 프로젝트 개요
|---|---|
| 항목 | 내용 |
| 프로젝트 이름 | **아바타 커마 log** |
| 한 줄 소개 | 인기 많은 아바타를 확인하고 여러 아바타와 이미지를 남길수 있는 아바타와 아바타 시세를 검색할수 있는 서비스 |
| 개발 기간 | 2026년 3월 ~ 7월 (1학기) |
| 배포 URL| Vercel을 통해 배포 예정 |

## 대상 사용자
- 유행중인 아바타를 찾고 있는 사람
- 자신만의 아바타 조합을 커스텀마이징 하고 싶은 사람
- 아바타의 시세를 확인하고 싶은 사람

## 핵심 기능

### 필수 기능

- [] **아바타 검색** - 아바타의 이름 혹은 관련 검색어 등으로 검색하고 결과를 목록으로 표시
- [] **아바타 직업군 분리** - 인게임 내의 직업별로 착용할수 있는 아바타가 다르기 때문에 직업별고 구분할 필요가 있음
- [] **커스텀 마이징 저장 기능** - 여러 아바타 파츠들을 모아 자신의 캐릭터에 커스텀마이징을 한 다음 저장할수 있는 기능 필요
- [] **아바타 이미지 정보 기능** - 아바타를 착용한 모습을 미리 볼수 있는 기능
- [] **아바타 시세 검색** - 자신이 검색한 아바타를 현재 구할수 있는지, 가격은 얼마인지 결과를 목록으로 표시

### 선택 기능

- [] 커스텀 마이징 삭제 기능
- [] 커스텀 마이징 수정 기능

## 기술 스택

|---|---|
| 기술 | 이유 |
| HTML | 웹사이트의 기본 구조 작성, 아바타 URL 입력 구조 |
| CSS | 웹사이트의 데이터 정리 및 시각화 |
| JS | 웹사이트의 기능 부여 |
| API | [DF_API_Docs](https://developers.neople.co.kr/contents/apiDocs/df) 게임 백엔드 데이터 추출|
| VS Code | 코드 작성 툴 |

## 페이지 구성

    1. 메인 페이지 아바타 및 캐릭터 선택이 가능한 검색 창, 랭킹 화면 이동 버튼
    2. 검색 결과 페이지 
        검색한 아바타를 적용한 캐릭터 
        아바타의 시세
        검색한 아바타의 점유율
    3. 랭킹 화면 이동이 동작 되는 버튼을 누를시 아바타의 랭킹 목록 나열이 된 페이지

## 개발 일정
| 기간 | 일정 | 상세 활동 내용 |
|...|...|...|
| 2~3 주차 | 기획 설계 및 프론트엔드 기초 학습 | 화면 설계 및 웹 기본 구조(HTML/CSS/JS) 학습
| 4~5 주차 | 프론트엔드 레이아웃 구현 | 웹 페이지의 메인 뼈대 및 결과 페이지의 시각적 뼈대 설계 |
| 6~7 주차 | JS를 이용한 기능 공부와 데이터 추출 로직 학습 | JS를 이용해 설계된 페이지에 기능 부여 및 Python 기초 문법 습득 및 DF_API 추출 방식 탐색 |
| 8~9 주차 | 본격적인 API 적용 | DF_API 에서 가져온 백엔드 데이터를 설계한 프론트엔드 웹사이트에 적용 |
| 10~11 주차 | 시스템 통합 및 시각화 | Python 서버와 웹 연결 및 JS를 활용한 분석 결과 시각화 구현 |
| 12~13 주차 | 단위 테스트 및 디버깅 코드 최적화 | 이미 생성되어 있는 다른 아바타 분석 웹사이트를 확인해보며 비교 및 코드 재검토 후 최적화 작업 |
| 14~15 주차 | 최종 보안 및 발표 | 서비스 최종 점검 및 프로젝트 결과 보고서/발표 자료 마무리
